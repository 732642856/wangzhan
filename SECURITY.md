# Security Policy

## Supported Versions

The `main` branch is the supported development line.

## Reporting

Please open a private report through GitHub Security Advisories if available, or contact the maintainer directly.

Do not post secrets, API keys, private scripts, or unpublished writing projects in public issues.

## Data Model

The hosted app is static and stores project data in the user's browser. Optional model-provider keys should stay local and must not be committed.
