# Contributing

Thanks for improving Personal Screenwriter.

## Setup

```bash
npm ci
npm run dev
```

## Before Pull Requests

```bash
npm test
npm run build
```

## Rules

- Keep private scripts, drafts, API keys, and ebook/OCR full text out of git.
- Prefer small focused changes.
- Add or update tests when changing user-visible behavior.
- For UI parity work, verify the page in a browser, not only by token checks.
